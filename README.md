# Proyecto Java – Compilación y Ejecución sin IDE

## Descripción del proyecto

Este proyecto consiste en el desarrollo de un programa en Java que permite ingresar datos por teclado y mostrarlos en pantalla. Se trabajó sin utilizar un entorno de desarrollo (IDE), utilizando la línea de comandos.

## Funcionalidades

- Ingresar nombre
- Ingresar edad
- Mostrar datos en pantalla

## Requerimientos funcionales

- Permitir ingreso de datos
- Mostrar los datos ingresados
- Ejecutarse desde consola

## Requerimientos no funcionales

- Código simple y entendible
- Ejecución sin errores
- Uso de Java sin IDE

## Instrucciones de uso

Compilar:
javac Main.java

Ejecutar:
java Main

## Tecnologías

- Java
- JDK
- JVM
