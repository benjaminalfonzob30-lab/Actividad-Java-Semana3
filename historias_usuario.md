# Historias de Usuario

## Historia 1
**Como** usuario  
**Quiero** ingresar mi nombre  
**Para** que el sistema pueda mostrarlo en pantalla  

**Criterio de aceptación:**  
- El sistema solicita el nombre  
- El usuario lo ingresa correctamente  
- El sistema lo muestra en pantalla  

---

## Historia 2
**Como** usuario  
**Quiero** ingresar mi edad  
**Para** que el sistema pueda mostrarla  

**Criterio de aceptación:**  
- El sistema solicita la edad  
- El usuario ingresa un número  
- El sistema muestra la edad correctamente  

---

## Historia 3
**Como** usuario  
**Quiero** ver los datos que ingresé  
**Para** confirmar que la información es correcta  

**Criterio de aceptación:**  
- El sistema muestra nombre y edad  
- La información coincide con lo ingresado
